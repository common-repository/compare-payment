== Description ==
Compare Payment plugin provides compares payment transaction amount between payment types. Compare Payment allows you to create unique payment type for displaying Payment Items consisting of screenshots and additional information such as payment logo, payment currency, payment transaction amount. 

= Features of the plugin includes =
* Choose Own Payment Method
* Transaction Fee
* Additional Fee
* Set Range Amount - Minimum and Maximum 
* Short-Code Support
* Separate post type for compare payment
* Translation ready
* Easy to use and customize
 
== Installation ==

1. First download the plugin ZIP file from wordpress.org
2. Log in to your website administrator panel
3. Go to the 'Add New' in the plugins dashboard, click “Upload Plugin”
4. Upload [Compare Payment](https://wordpress.org/plugins/compare-payment/) ZIP file by choosing it from your download location
5. Click Install Now button
6. Then click Activate the Plugin button.
7. You can see the plugin installed on Wordpress left menu.
8. Include shortcode in your in any content. e.g. post, page, theme

= How to use =
1. First installed and active the plugin
2. Create a new Payments Type
3. Include shortcode in your in any content. e.g. post, page, theme
4. Set Default Options from Comapare payment Menu

### Use Shortcode in Post or Page:-
<code>[compare-payment]</code>
